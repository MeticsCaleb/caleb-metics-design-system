## House defaults (Caleb / Metics Media — these override anything below)

This channel leads with full-frame motion graphics, not talking-head overlays. Apply these as the governing defaults; where they conflict with anything later in this document, these win.

- Full-frame first. SOLID full-frame graphics on a white/light base are the DEFAULT clip type. Use ALPHA talking-head overlays only at beats explicitly marked talking-head (typically the open and the CTA).
- No reserved presenter zone. Do not keep the center cells or the bottom-right corner clear for the presenter. Graphics use the entire frame, content centered. - Lower-third: "Caleb" with no trailing period; wordmark set as two words, "Metics Media."
- Link / CTA card: a white watch-card that pops up from the bottom, full width, covering the bottom quarter of the frame (light-mode YouTube styling, link in YouTube's light-mode blue). It can hold through the outro crossfade.
- Keep it calm: one idea per slide, icon-and-number led, centered, never fast-paced.
